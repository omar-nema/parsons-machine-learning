Explore the Palestine Poster Archive
In this application, a set of 5,000 posters created by Palestinian artists, and drawn from the Palestine Poster Project archive, are sorted by similarity using machine learning.

Poster assets and metadata were taken from the Palestine Poster Project, curated by Dan Walsh. The PPP archive contains a set of 10,000 posters dating from 1930 to present day. Sole credit for archiving and categorizing the posters in this project goes to the PPP team (I had no part in this work).
In the grid below, posters are sorted by similarity - posters close to each other share similar visual content, iconography, and artists (determined by a combination of machine learning methods - feature extraction and dimensional reduction).

Scroll to zoom in and out, click on a poster to view details.
