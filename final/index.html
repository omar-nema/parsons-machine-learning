Explore the Palestine Poster Archive
In this application, a set of 5,000 posters drawn from the Palestine Poster Project archive are sorted by similarity using machine learning. The result is a canvas that allows you to explore artifacts of Palestinian identity and resistance over the past 85 years.

Note that this app only displays posters created by Palestinian artists. Non-Palestinian Arab, and Israeli artists were excluded. Please visit the Palestine Poster Project if you're curious to explore posters in these domains.

Poster assets and metadata were taken from the Palestine Poster Project, curated by Dan Walsh. The PPP archive contains a set of 10,000 posters related to Palestine dating from 1933 to present day. Sole credit for archiving and categorizing the posters in this project goes to the PPP team.
